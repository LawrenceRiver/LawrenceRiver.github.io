# 页面设计文档（Desktop-first）

## 全局设计规范
### Layout
- 桌面优先：内容最大宽度 1080–1200px 居中；大区块上下间距 48–64px。
- 栅格：列表页使用 12 栅格或 CSS Grid；卡片列表桌面 3 列、平板 2 列、移动 1 列。
- 响应式：断点建议 `lg(≥1024) / md(≥768) / sm(<768)`；导航在 `sm` 折叠为抽屉/汉堡菜单。

### Meta Information（通用策略）
- Title：`{页面名} - {你的站点名}`；详情页为 `{条目标题} - {站点名}`。
- Description：列表页为该内容线简介；详情页使用条目 summary。
- Open Graph：`og:title/og:description/og:image/og:type=website`；详情页 type 仍为 website（静态站点）。

### Global Styles（Design Tokens）
- 背景：#0B0F14（深色）或 #FFFFFF（浅色）二选一；建议提供主题切换（可选，不作为必需）。
- 主色（Accent）：#6D5EF7（偏科技紫）
- 文字：
  - 标题：H1 36/44、H2 28/36、H3 20/28
  - 正文：16/26；辅助文字 14/22
- 卡片：圆角 12px；边框 1px（浅灰/深灰）；hover 提升阴影或边框高亮。
- 按钮：Primary/Secondary/Link 三种；hover 颜色加深；disabled 降低不透明度。
- 链接：默认下划线弱化；hover 显示下划线或变色。

### 通用组件
- Top Nav：站点名/Logo、主导航（首页/歌手/科研）、全站搜索框（桌面）、外链（可选：GitHub/社媒）。
- Footer：版权、联系邮箱（可选）、站点构建信息（可选）。
- Content Card：封面（可选）、标题、时间、标签、summary、进入详情按钮/整卡可点击。
- Tag/Filter Bar：年份、标签、多选；排序下拉。

## 页面 1：首页（/）
### Page Structure
- 顶部导航（Top Nav）
- Hero（站点定位 + 双线入口 CTA）
- 双线概览区（两张大卡：歌手内容 / AI 科研）
- 置顶推荐（可配置，最多 3–6 条）
- 最新更新（跨类型聚合流）
- Footer

### Sections & Components
1) Hero
- 左侧：一句话定位（例如“歌手记录与 AI 科研笔记”）
- 右侧：两个主按钮（进入歌手内容/进入 AI 科研）

2) 双线概览
- 两张对称信息卡：
  - 卡片标题、简介、近期更新摘要（1–3 条）
  - “查看全部”按钮跳转对应列表页

3) 置顶推荐
- 卡片网格，支持置顶标识（Badge：Pinned）
- 点击进入详情页

4) 最新更新
- 时间线或列表（推荐时间线），每条显示：类型徽标（论文/专辑/周边/巡演）、标题、更新时间

5) 全站搜索（与 Top Nav 搜索联动）
- 输入即过滤（前端本地索引）
- 结果项点击进入详情页

## 页面 2：歌手内容页（/singer）
### Page Structure
- Top Nav
- 页面标题区（标题 + 简介）
- 类型切换 Tabs（专辑/周边/巡演）
- 筛选与排序条（年份/标签/在售/城市等，按类型变化）
- 内容卡片网格
- Footer

### Sections & Components
1) Tabs（专辑/周边/巡演）
- 选中态明显；切换时保持筛选条与列表同步更新

2) Filter Bar（随类型变化）
- 专辑：年份、标签、平台（可选）
- 周边：标签、在售状态（可选）
- 巡演：年份、城市、时间排序（即将开始优先）

3) 列表卡片
- 专辑卡：封面、标题、发行日期、标签、summary
- 周边卡：图、标题、状态（在售/售罄）、summary
- 巡演卡：日期、城市/场馆、购票按钮（若有链接）

交互
- 卡片点击进入详情页
- 列表空状态：显示“暂无内容/调整筛选条件”

## 页面 3：AI 科研内容页（/research）
### Page Structure
- Top Nav
- 页面标题区（标题 + 简介）
- 筛选与排序条（年份/会议期刊/标签）
- 论文列表（更适合列表而非大图卡）
- Footer

### Sections & Components
1) Filter Bar
- 年份下拉 + 会议/期刊下拉 + 标签多选
- 排序：按年份（降序默认）/按更新时间

2) 论文条目（List Item）
- 标题（主）
- 作者、会议/期刊、年份（次要信息行）
- 标签（chips）
- 摘要（1–3 行截断）
- 右侧快捷外链按钮：PDF / Code（若存在）

交互
- 点击标题或整条进入详情页

---

## 页面 4：内容详情页（/item/:type/:slug）
### Page Structure
- Top Nav
- 面包屑（首页 > 歌手/科研 > 类型）
- 标题区（标题、时间、标签、关键元信息）
- 主内容区：正文（Markdown 渲染）+ 侧栏（关键信息与外链）
- 同类跳转（上一条/下一条或“返回列表”）
- Footer

### Sections & Components
1) 标题区
- 标题
- Meta 行：
  - 论文：作者、venue、year
  - 专辑：releaseDate
  - 周边：availabilityStatus（可选）、price（可选）
  - 巡演：eventDate、city、venue

2) 正文内容（Markdown）
- 支持：标题层级、列表、引用、图片、代码块（科研页更常用）
- 图片：点击可放大（可选）

3) 侧栏（Actions）
- 外链按钮组（根据内容类型显示）：PDF、Code、购买、购票、平台收听等
- 分享：复制链接（可选）

4) 返回与同类导航
- 固定提供“返回列表”
- 可选提供上一条/下一条（按当前排序规则）
